# readability-cli

Extract article content from a URL and convert it to Markdown.

## Usage

### URL

```bash
bun run index.ts --url <URL>
```

### File

```bash
bun run index.ts --file <path/to/file.html>
```

### Directory

```bash
bun run index.ts --dir <path/to/directory>
```

## Build

```bash
bun run build
```

## Run Binary

```bash
./dist/readability-cli --url <URL>
./dist/readability-cli --file <path/to/file.html>
./dist/readability-cli --dir <path/to/directory>
```
